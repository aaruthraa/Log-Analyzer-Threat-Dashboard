# Log Analyzer and Threat Dashboard

## Description

A Flask-based web application that analyzes server log files and detects suspicious activities such as failed login attempts and brute-force attacks.

## Features

- Upload log files
- Parse server logs
- Detect failed login attempts
- Detect brute-force attacks
- Dashboard with statistics
- Pie chart and bar chart
- Search by IP address
- Risk classification of suspicious IPs

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Chart.js

## Installation

```bash
pip install -r requirements.txt
python app.py
```

Open:

```
http://127.0.0.1:5000
```
